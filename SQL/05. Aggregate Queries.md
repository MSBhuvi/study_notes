# Aggregate Function in SQL

An aggregate function in SQL is a function that performs a calculation on multiple rows of a table and returns a single summarized value.

Aggregate functions are mainly used for data analysis, reporting, and summarization.

## Common Aggregate Functions

| Function  | Description           | Example Use    |
| --------- | --------------------- | -------------- |
| `COUNT()` | Counts number of rows | Total records  |
| `SUM()`   | Adds numeric values   | Total salary   |
| `AVG()`   | Calculates average    | Average marks  |
| `MIN()`   | Finds smallest value  | Lowest price   |
| `MAX()`   | Finds largest value   | Highest salary |

### Aggregate Function Practice Queries

**1. Total number of orders**

```
SELECT COUNT(*) AS total_orders FROM orders;
```

**2. Total quantity sold**

```
SELECT SUM(quantity) AS total_quantity FROM orders;
```

**3. Average price of products**

```
SELECT AVG(price) AS avg_price FROM orders;
```

**4. Maximum price**

```
SELECT MAX(price) AS max_price FROM orders;
```

**5. Minimum price**

```
SELECT MIN(price) AS min_price FROM orders;
```

**6. Category-wise total sales amount**

```
SELECT category, SUM(quantity * price) AS total_sales
FROM orders
GROUP BY category;
```

**7. Category-wise order count**

```
SELECT category, COUNT(order_id) AS order_count
FROM orders
GROUP BY category;
```

**8. Average price per category**

```
SELECT category, AVG(price) AS avg_price
FROM orders
GROUP BY category;
```

**9. Categories with total sales above 50,000**

```
SELECT category, SUM(quantity * price) AS total_sales
FROM orders
GROUP BY category
HAVING SUM(quantity * price) > 50000;
```
