# Aggregate Function in SQL

An aggregate function in SQL is a function that performs a calculation on multiple rows of a table and returns a single summarized value.

Aggregate functions are mainly used for data analysis, reporting, and summarization.

## Common Aggregate Functions

| Function  | Description           | Example Use    |
| --------- | --------------------- | -------------- |
| `COUNT()` | Counts number of rows | Total records  |
| `SUM()`   | Adds numeric values   | Total salary   |
| `AVG()`   | Calculates average    | Average marks  |
| `MIN()`   | Finds smallest value  | Lowest price   |
| `MAX()`   | Finds largest value   | Highest salary |
