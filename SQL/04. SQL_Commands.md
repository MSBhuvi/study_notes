## 1. Data Definition Language (DDL) Commands

DDL commands define or modify the structure of a database (tables, schemas, etc.). They permanently affect the database structure.

### a. CREATE

Used to create databases, tables, indexes, views, etc.

```
CREATE TABLE employees (
    id INT PRIMARY KEY,
    name VARCHAR(50),
    salary DECIMAL(10,2)
);
```
### b. ALTER

Used to modify an existing table structure (add, drop, or modify columns).

```
ALTER TABLE employees ADD age INT;
```

### c. DROP

Deletes an entire table, database, or object permanently.

```
DROP TABLE employees;
```

### d. TRUNCATE

Removes all rows from a table but keeps the structure.

```
TRUNCATE TABLE employees;
```

## 2. Data Manipulation Language (DML) Commands

DML commands deal with inserting, modifying, or deleting data inside the tables.

### a. INSERT

Adds new records into a table.

```
INSERT INTO employees (id, name, salary)
VALUES (1, 'Bhuvi', 50000);
```

### b. UPDATE

Modifies existing records.

```
UPDATE employees
SET salary = 60000
WHERE id = 1;
```

### c. DELETE

Removes specific records from a table.

```
DELETE FROM employees WHERE id = 1;
```
